# Hexagonal Version Archive

이 폴더는 MVC 선택 이전에 작성된 Lightweight Hexagonal 산출물을 보존한다.

## Status

- Active architecture: `planning-artifacts/mvc-selection.md`
- Active planning artifacts: `planning-artifacts/`
- Active implementation status: `implementation-artifacts/`
- Archived architecture: Lightweight Hexagonal

## Use

- 제품 문제, 계약 변환 근거, 과거 판단을 비교해야 할 때만 참고한다.
- 새 story 구현, sprint planning, BMAD dev story 컨텍스트에서는 active MVC 산출물을 기준으로 삼는다.
- Hexagonal package boundary, port/adapter 구조, `domain/application` layering은 더 이상 구현 기준이 아니다.
